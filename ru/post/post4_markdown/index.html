Examples

Code

Wowchemy поддерживает расширение Markdown для выделения синтаксиса кода. Вы можете настроить стили с помощью параметра “syntax_highlighter” в вашем файле “config/_default/params.yaml”.

```python
import pandas as pd
data = pd.read_csv("data.csv")
data.head()
```

визуализируется так

import pandas as pd
data = pd.read_csv("data.csv")
data.head()

Mindmaps

Wowchemy поддерживает расширение Markdown для mindmaps.

Просто вставьте блок кода Markdown markmap и при желании установите высоту mindmap, как показано в примере ниже.

A simple mindmap defined as a Markdown list:


```markmap {height="200px"}
- Hugo Modules
  - wowchemy
  - blox-plugins-netlify
  - blox-plugins-netlify-cms
  - blox-plugins-reveal
```

визуализируется так

- Hugo Modules
  - wowchemy
  - blox-plugins-netlify
  - blox-plugins-netlify-cms
  - blox-plugins-reveal

    -
      ```js
      console.log('hello');

Charts

Wowchemy поддерживает популярный формат интерактивных графиков Plotly.

Сохраните свой графический JSON-файл в папке page, например “line-chart.json”, а затем добавьте короткий код {{< chart data="line-chart" >}} в том месте, где вы хотели бы видеть диаграмму.

Math

Wowchemy поддерживает расширение Markdown для $\LaTeX$ math. Вы можете включить эту функцию, переключив параметр math в config/_default/params.yaml.

Чтобы отобразить inline or block math, берните математику в LaTeX с помощью {{< math >}}$...${{< /math >}} or {{< math >}}$$...$${{< /math >}}, соответственно.

Example math block:

{{< math >}}
$$
\gamma_{n} = \frac{ \left | \left (\mathbf x_{n} - \mathbf x_{n-1} \right )^T \left [\nabla F (\mathbf x_{n}) - \nabla F (\mathbf x_{n-1}) \right ] \right |}{\left \|\nabla F(\mathbf{x}_{n}) - \nabla F(\mathbf{x}_{n-1}) \right \|^2}
$$
{{< /math >}}

визуализируется так

$$\gamma_{n} = \frac{ \left | \left (\mathbf x_{n} - \mathbf x_{n-1} \right )^T \left [\nabla F (\mathbf x_{n}) - \nabla F (\mathbf x_{n-1}) \right ] \right |}{\left \|\nabla F(\mathbf{x}_{n}) - \nabla F(\mathbf{x}_{n-1}) \right \|^2}$$

Example inline math {{< math >}}$\nabla F(\mathbf{x}_{n})${{< /math >}} renders as $\nabla F(\mathbf{x}_{n})$ .

Example multi-line math using the math linebreak (\\):

{{< math >}}
$$f(k;p_{0}^{*}) = \begin{cases}p_{0}^{*} & \text{if }k=1, \\
1-p_{0}^{*} & \text{if }k=0.\end{cases}$$
{{< /math >}}

визуализируется так

$$ f(k;p_{0}^{*}) = \begin{cases}p_{0}^{*} & \text{if }k=1, \\ 1-p_{0}^{*} & \text{if }k=0.\end{cases} $$

Diagrams

Wowchemy поддерживает расширение Markdown для диаграмм. Вы можете включить эту функцию, переключив параметр “диаграмма” в вашем файле “config/_default/params.toml” или добавив “диаграмма: true” в заголовок вашей страницы.

An example flowchart:

```mermaid
graph TD
A[Hard] -->|Text| B(Round)
B --> C{Decision}
C -->|One| D[Result 1]
C -->|Two| E[Result 2]
```

визуализируется так

graph TD A[Hard] -->|Text| B(Round) B --> C{Decision} C -->|One| D[Result 1] C -->|Two| E[Result 2]

An example sequence diagram:

```mermaid
sequenceDiagram
Alice->>John: Hello John, how are you?
loop Healthcheck
    John->>John: Fight against hypochondria
end
Note right of John: Rational thoughts!
John-->>Alice: Great!
John->>Bob: How about you?
Bob-->>John: Jolly good!
```

визуализируется так

sequenceDiagram Alice->>John: Hello John, how are you? loop Healthcheck John->>John: Fight against hypochondria end Note right of John: Rational thoughts! John-->>Alice: Great! John->>Bob: How about you? Bob-->>John: Jolly good!

Todo lists

Вы даже можете создавать свои списки дел в Markdown:

- [x] Write math example
  - [x] Write diagram example
- [ ] Do something else

отображается так

  • Write math example
    • Write diagram example
  • Do something else

{{< table path="results.csv" header="true" caption="Table 1: My results" >}}

Callouts

Academic поддерживает shortcode for callouts, также называемый as asides, hints, or alerts. Заключая абзац в {{% callout note %}} ... {{% /callout %}}, это будет отображаться как отступление.

{{% callout note %}}
A Markdown aside is useful for displaying notices, hints, or definitions to your readers.
{{% /callout %}}

визуализируется так

Markdown полезен для отображения уведомлений, подсказок или определений для ваших читателей.

Spoilers

Добавьте спойлер на страницу, чтобы после нажатия кнопки отображался текст, например ответ на вопрос.

{{< spoiler text="Click to view the spoiler" >}}
You found me!
{{< /spoiler >}}

визуализируется так

Click to view the spoiler
You found me!

Вот несколько примеров использования шорткода “icon” для отображения значков:

{{< icon name="terminal" pack="fas" >}} Terminal  
{{< icon name="python" pack="fab" >}} Python  
{{< icon name="r-project" pack="fab" >}} R
